
# Предвестник — Telegram бот карты дня

## Что умеет
- Карта дня
- Перевёрнутые карты
- Случайные трактовки
- Без повторов трактовок
- Кнопка доната
- Кнопка канала

---

## 1. Установка Python
Скачать:
https://www.python.org/downloads/

---

## 2. Скачать проект

Распаковать архив.

---

## 3. Установить зависимости

В терминале:

pip install -r requirements.txt

---

## 4. Создать .env

Скопировать .env.example в .env

Вставить:
- токен бота
- ссылки

---

## 5. Запуск

python main.py

---

## 6. Как подключить к каналу

1. Открыть Telegram канал
2. Управление каналом
3. Администраторы
4. Добавить администратора
5. Добавить своего бота
6. Выдать права:
   - публикация сообщений

---

## 7. Как добавить новые трактовки

Открыть:

data/cards.json

Добавлять новые карты и тексты по аналогии.

---

## 8. Где разместить бота 24/7

Рекомендуется:
https://timeweb.cloud/

Минимальный сервер:
- Ubuntu 22
- 1 GB RAM

---

## 9. ВАЖНО

Никогда не публикуйте токен бота в открытом доступе.
